задача клубов

Создать «интеллектуальное ядро» для поиска прорывных идей по социально-экономическому, инновационному и инвестиционному развитию России, внедрения лучших практик и совместной реализации проектов. Формат клуба позволяет всем заинтересованным сторонам разрабатывать стратегии развития территорий.

Задача — отбор, экспертиза и реализация проектов для регионального роста. Через клуб ОП АСИ и РОИВ подключаются к организации форума «Сильные идеи для нового времени».
